# Nama Aplikasi
Kalender Pengingat Kegiatan

# Deskripsi Aplikasi
Aplikasi ini menggunakan API kalender. Dalam aplikasi ini user dapat mencatat kegiatan pada hari tertentu dan dapat diatur alarm ketika kegiatan telah selesai.
